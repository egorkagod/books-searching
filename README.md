# Book Searching

Сервис подбора книг с использованием семантического поиска.

## Описание

Web-приложение с чат-интерфейсом, которое позволяет находить книги по описанию на естественном языке.  
Сервис использует векторный поиск для определения смысловой близости между запросом пользователя и книгами, что позволяет находить релевантные результаты даже без точных совпадений по ключевым словам.

## Основной функционал

- Загрузка и индексация каталога книг (JSON / форма)
- Преобразование текстовых описаний в векторные представления (embeddings)
- Семантический поиск книг по пользовательскому запросу
- Интерфейс для взаимодействия с системой
- Выдача топ-5 наиболее релевантных книг с кратким описанием

## Технологии

- Backend: FastAPI  
- Векторная БД: Milvus  
- Модель эмбеддингов: BAAI/bge-m3  
- Контейнеризация: Docker, docker-compose  

## Требования

- Docker
- Docker Compose

## Запуск проекта

```bash
git clone https://github.com/egorkagod/books-searching.git
cd books-searching
docker compose up --build
```

## Загрузка данных

После запуска база данных не содержит книг.  
Для работы поиска необходимо загрузить каталог книг.

Это можно сделать через веб-интерфейс по адресу http://localhost:8080

После загрузки данные индексируются и сохраняются в векторной базе (Milvus).
