## Задание: реализовать Web-приложение для подбора книг

### Цель
Создать веб-приложение с чат-интерфейсом для поиска книг по описанию на естественном языке с использованием векторного поиска.

### Исходные данные
- Предоставляется файл `books.json` с каталогом книг и их метаинформацией

### Технические требования

#### Архитектура и развертывание
1. Приложение должно развертываться с помощью Docker и docker-compose
2. Все компоненты должны быть описаны в одном compose-файле

#### Компоненты системы
1. **Векторная база данных**: любая open-source БД (например: Qdrant, Milvus)
2. **Embedding модель**: любая малая модель для создания векторных представлений (например: BAAI/bge-m3)
3. **Backend**/**Frontend**: можно реализовать на любом удобном web-фреймворке (например: Streamlit, FastAPI)

### Функциональные требования

#### Управление данными
- Возможность загрузки каталога книг через веб-интерфейс
- Индексация загруженных книг в векторной БД

#### Поиск книг
- Чат-интерфейс для ввода запросов на естественном языке
- Поиск по семантическому сходству
- Возврат топ-5 наиболее релевантных книг

#### Пример использования
**Запрос пользователя**: "Роман с неожиданной развязкой"
**Ответ системы**: Список из 5 наиболее подходящих книг с краткой информацией
